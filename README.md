# 返信付き商品レビュープラグイン for EC-CUBE 4.0系

コミュニティで開発されている[商品レビュー管理プラグイン][productreview-plugin]をカスタマイズしたプラグインです。  
商品ページに商品のレビュー機能を追加する機能に加えて、レビューに対してお店から返信コメントができるようになっています。
